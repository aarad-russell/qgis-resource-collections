# QGIS Resource Collections
A repository for QGIS styles, icons and other resource types that can be shared with the QGIS Resource Sharing Plugin: https://plugins.qgis.org/plugins/qgis_resource_sharing/

1. Install the QGIS Resource Sharing Plugin
2. Add this repository
   - Instructions: https://qgis-contribution.github.io/QGIS-ResourceSharing/usage/adding-repository.html
   - Repository URL: https://github.com/aarad-russell/qgis-resource-collections.git
3. Install the desired collections.

# Collections
## New Zealand Search and Rescue
This provides a collection of resources for managing and styling New Zealand search and rescue data in QGIS.
- Symbols
  - CIMS roles
  - Common search point markers
  - Search tracks for commonly used resource types/organisations
 
  _This has no association with the NZSAR organisation (https://nzsar.govt.nz/). This is not an officially recognised symbology set._
